## ğŸš€ Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/iromn/olirvu.git
    cd olirvu
    ```

2. **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    ```

### Running the App

Start the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:5173](http://localhost:5173) (or the port shown in your terminal) to view the app in the browser.

### Building for Production

To build the app for production:

```bash
npm run build
```

This will create a `dist` folder with the optimized assets ready for deployment.
